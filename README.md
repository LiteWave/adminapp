# adminapp

The admin app for Litewave

## Setup

Install Node Virtual Environment 

`sudo pip install nodeenv`
`nodeenv env`

Activate your nodeenv:

`. env/bin/activate`

Install the npm plugins:

`npm install`

Install grunt:

`npm install -g grunt`


## Running locally

`grunt server`

Server available at:

`http://localhost:9000/`

## Deployment

`grunt deploy:prod`

Server available at:

`www.litewaveinc.com/admin`
