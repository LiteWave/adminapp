Use this page to create a list of groups for a stadium

To create a layout, follow these steps
  • In the Stadium map click on all the sections to include in this group.
  • Once you have all the sections selected, press the Create Group button. The new group will appear below the Stadium map.
  • Click Save Layout after you have all of the sections included in groups.
